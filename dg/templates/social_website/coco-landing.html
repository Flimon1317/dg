COCO

Connect Online | Connect Offline

COCO is a client side framework that enables data entry in intermittent Internet connectivity. Data can be stored locally while the user is offline, and when he comes back online, the data can be synced to the server.

Developed as a Framework

COCO v2 is only a client side framework which provides a single page application. It is a collection of HTML CSS and Javascript files, which can be easily plugged into a database.

Naturally Versatile

COCO is designed to be completely platform independent, meaning that it can be used with a variety of back-end implementations. It only requires REST functionality to be implemented on the database.

Easily Adaptable

COCO has been coded in a way that it can adapt to various needs. All kinds of data can be entered, edited and viewed through the framework. You dont need to edit/customise any view, it will simply take the parameters as input and adapt accordingly.

Responsive to screen size

Coco is adaptable to all screen sizes, enabling easy use on smaller devices like tablets and smartphones. You don't need to do anything at all, its already built in.

Searching and sorting

The listviews in Coco come with built in searching and sorting functionality that make things so much easier to find.

Data in your format

The data in Coco's listviews can be downloaded as excel files or copied to clipboard with just a click of a button. Its so easy.
