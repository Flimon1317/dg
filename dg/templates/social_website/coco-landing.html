COCO

Connect Online | Connect Offline

COCO is a client side framework that enables data entry in intermittent Internet connectivity. Data can be stored locally while the user is offline, and when he comes back online, the data can be synced to the server.

SYNC

Sync to Server when online

Search and Sort

Sync to Server when online

The listviews in Coco come with built in searching and sorting functionality that make things so much easier to find.

Download Data

Download Data in Excel format

The data in Coco's listviews can be downloaded as excel files or copied to clipboard with just a click of a button. Its so easy.

Adaptable

Responsive to screen size

Coco is adaptable to all screen sizes, enabling easy use on smaller devices like tablets and smartphones. You don't need to do anything at all, its already built in.
